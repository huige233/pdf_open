## 名字(Name)

pdf_open -- 一款简单且极少依赖的pdf拆解程序

## 使用(Useage)

需要python3
需要下载fitz库
```cmd
pip install fitz
```
若还是报错 请安装PyMuPDF
```cmd
pip install PyMuPDF
```

如果不想修改程序请将pdf命名为1.pdf放在同目录文件夹下
若有修改需求请在__main__中修改
